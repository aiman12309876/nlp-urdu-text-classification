[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)

## Overview

A machine learning system for classifying Urdu news headlines into categories like Politics, Sports, Entertainment, Business, Technology, and Health. This project demonstrates NLP techniques for low-resource languages, specifically Urdu.

## Why This Project Matters

- Urdu is the national language of Pakistan
- Low-resource language NLP is a critical research area
- Demonstrates handling of non-Latin script
- Real-world application for news categorization
